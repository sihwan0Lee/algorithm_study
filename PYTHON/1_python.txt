파이썬

- 특징
Easy to learn
Easy to read
Easy to maintain
1. 인터프리터식
2. 객체지향적
3. 동적 타이핑 대화형 언어 (dynamic typing)
4. 모듈, 클래스, 객체와 같은 언어의 요소가 내부에서 접근할 수 있다.
5. 들여쓰기(indent)를 통한 블록 구분


- 장점
1. 라이브러리(모듈)이 풍부하다.
2. 순수한 프로그램 언어로서의 기능 외에도 다른 언어로 쓰인 모듈들을 연결하는 접착언어로 자주 이용된다.
3. Pyrex, Psyco, Numpy 등을 이용하여 수학, 과학 공학 분야등에서 많이 활용 되고 있다.
- 단점
1. C/C++ 등의 언어에 비해 수행 속도가 느리다.(하지만 서비스, 일반적인 컴퓨터 응용환경에서는 속도가 크게 중요하지 않으며,
빠른 속도를 요구하는 프로그램의 경우 프로토타이핑한 뒤 빠른 속도가 필요한 부분만 C 언어등으로 모듈화 할수 있다고 한다.ctypes, SWIG, SIP)

+ 특징 설명 주석
1. 인터프리터 언어 (스크립트 언어)
실행이 될 때 마다 바로 해석해서, 코드 변경시 실행때 마다 결과가 바뀐다.

2. 객체 지향 언어
데이터를 객체 취급해서 필요 기능들을 하나의 객체로 묶어 프로그래밍하는 언어이다.
기능을 담은 함수와 필드 등으로 묶어 하나의 클래스를 만든다.
객체 지향 언어는 모듈화가 되어있기 때문에 유지/보수에 좋다.
객체가 곧 하나의 프로그램이기 떄문에 상속등과 같이 코드 재사용이 용이하다.
하지만 많은 양의 메모리를 사용하기 떄문에 설계에 주의해야한다.
포인트는 함수와 클래스를 통해 기능을 객체로 묶어 프로그래밍 한다는 것이다.
